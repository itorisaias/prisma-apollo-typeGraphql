# GraphQL

## :rocket: Technologies

-  Apollo server
-  Prisma
-  TypeGraphQL

## :information_source: How To Use

Clone the repo
  ```sh
  git clone https://github.com/viniciushvc/prisma-apollo-typeGraphql
  ```

Install packages
  ```sh
  npm install
  ```

Rename .env.local.example to
  ```sh
  .env.local
  ```

Start Postgresql
  ```sh
  make up
  ```

Run migrate
  ```sh
  npm run migrate
  ```

Run generate
  ```sh
  npm run generate
  ```

Run project
  ```sh
  npm run dev
  ```
